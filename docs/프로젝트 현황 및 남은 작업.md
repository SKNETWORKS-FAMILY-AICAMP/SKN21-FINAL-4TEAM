## 4. 사용자가 직접 설정해야 할 사항

### 4.1 환경 변수 파일 생성 (필수)

`.env.example`을 복사하여 `.env` 파일을 생성하고 실제 값을 입력해야 합니다.

```bash
cp .env.example .env
```

#### 환경 변수 목록

| 변수 | 설명 | 기본값 / 비고 |
|---|---|---|
| **DATABASE_URL** | PostgreSQL 비동기 접속 URL | `postgresql+asyncpg://chatbot:chatbot@localhost:5432/chatbot` (Docker 기본값 사용 가능) |
| **DATABASE_SYNC_URL** | PostgreSQL 동기 접속 URL (Alembic용) | `postgresql://chatbot:chatbot@localhost:5432/chatbot` |
| **REDIS_URL** | Redis 접속 URL | `redis://localhost:6379/0` (Docker 기본값 사용 가능) |
| **SECRET_KEY** | JWT 서명용 시크릿 키 | 반드시 랜덤 문자열로 변경 (`openssl rand -hex 32`) |
| **ACCESS_TOKEN_EXPIRE_MINUTES** | JWT 토큰 만료 시간(분) | `60` |
| **RUNPOD_API_KEY** | RunPod API 키 | RunPod 대시보드에서 발급 |
| **RUNPOD_ENDPOINT_ID** | RunPod Serverless 엔드포인트 ID | 엔드포인트 생성 후 확인 |
| **OPENAI_API_KEY** | OpenAI API 키 | GPT-4o 사용 시 필요 |
| **ANTHROPIC_API_KEY** | Anthropic API 키 | Claude 사용 시 필요 |
| **GOOGLE_API_KEY** | Google AI API 키 | Gemini 사용 시 필요 |
| **LANGFUSE_PUBLIC_KEY** | Langfuse 퍼블릭 키 | 선택 (없으면 관측성 로깅 스킵) |
| **LANGFUSE_SECRET_KEY** | Langfuse 시크릿 키 | 선택 |
| **LANGFUSE_HOST** | Langfuse 호스트 URL | `http://localhost:3001` |
| **SENTRY_DSN** | Sentry DSN URL | 선택 (없으면 에러 추적 비활성) |
| **APP_ENV** | 앱 환경 | `development` |
| **DEBUG** | 디버그 모드 | `true` |
| **CORS_ORIGINS** | 허용 CORS 출처 | `http://localhost:3000` |

> **참고:** LLM API 키는 최소 1개 이상 설정해야 채팅 기능이 동작합니다. RunPod을 기본 모델로 사용하려면 RunPod 키가 필수이고, 외부 API(OpenAI/Anthropic/Google)를 사용하려면 해당 키를 설정하면 됩니다.
