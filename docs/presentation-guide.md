# 발표 PPT 제작 가이드

**발표 정보**
- 대상: 현업 개발자 · 심사위원 · 동료 개발자
- 목적: 교육과정 최종 프로젝트 발표
- 시간: 15분 발표 + 5분 Q&A
- 슬라이드: 총 17장 (예비 1장 포함)

---

## 디자인 시스템

```
배경색:     #0f172a  (진한 남색)
강조색 1:   #f97316  (오렌지 — 메인)
강조색 2:   #38bdf8  (하늘색 — 보조)
성공/긍정:  #4ade80  (초록)
카드 배경:  #1e293b  (중간 남색)
텍스트:     #ffffff  (흰색) / #94a3b8 (회색 — 보조 설명)

폰트:
  제목  → Pretendard Bold (없으면 Noto Sans KR Bold)
  내용  → Pretendard Regular
  크기  → 제목 36~54pt / 소제목 22~28pt / 본문 14~18pt

슬라이드 크기: 16:9 (와이드)
```

**Canva 색상 팔레트 저장 순서:**
1. canva.com → 새 디자인 → 프레젠테이션 (16:9)
2. 브랜드 킷 → 색상 추가 → 위 6개 색상 코드 입력
3. 배경 `#0f172a` → 슬라이드 전체 적용
