## 7. 선정 모델 특성 상세

### gpt-5-nano (Review 선정)

```
장점:
  - 가장 낮은 비용: $0.05/$0.40 per 1M tokens
  - 빠른 응답: 380ms (gpt-4o-mini 900ms의 42%)
  - GPT-5 아키텍처 기반 → 명령 추종 개선
  - 400K 컨텍스트 (gpt-4o-mini 128K의 3배)

단점:
  - 복잡한 허위 주장 탐지 정확도 gpt-4.1-mini 대비 소폭 낮음
  - 신규 모델 안정성 검증 기간 필요

완화:
  - Phase3 패스트패스: 복잡도 높은 발언만 LLM 검토 (80% 스킵)
  - regex 1차 필터 (프롬프트 인젝션, 인신공격) 선행
  - 최종 판정은 gpt-4.1이 전체 발언을 재검토
```

### gpt-4.1 (Judge 선정)

```
장점:
  - gpt-4o 대비 20% 저렴 ($0.01179 vs $0.01474)
  - 1040K 컨텍스트 → 긴 토론 히스토리 완전 수용
  - 향상된 명령 추종 → JSON 출력 안정성 상승
  - 검증된 출시 모델 (안정성 높음)

단점:
  - o3/gpt-5 대비 추론 깊이 소폭 낮음
  - 토론 특화 파인튜닝 없음 (범용 모델)

완화:
  - JUDGE_SYSTEM_PROMPT가 채점 기준 명확히 정의
  - 50% 스왑 편향 제거 로직으로 일관성 보완
  - 점수차 5점 미만 무승부 처리로 경계 케이스 안전 처리
```
